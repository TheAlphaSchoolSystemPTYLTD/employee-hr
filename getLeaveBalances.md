**getLeaveBalances**
----
  Returns an array of structured Leave Balances details in JSON format.

* **Version:**

  2

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**

   `code [string]` - Employee code
   
   **Optional:**

   none

   **Conditional:**
 
   none

    `code` not supplied
    ```javascript

    ```

