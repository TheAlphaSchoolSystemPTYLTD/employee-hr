# Employee HR API
This API extracts the following employee related information:
 
-    Employee Photo (if enabled in TASS.web)
-    HR General tab
-    Address tab
-    Next of Kin
-    Leave Balances
-    HR User Defined information


**Methods**

  * addEmployee
  * getEmployeesDetails
  * getEmployeeUD
  * getEmployeeUDSetup
  * getLeaveBalances

V2 and V3 versions of this API are available.
